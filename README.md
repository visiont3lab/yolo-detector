# YOLO Detector

## Setup

* Get yolo models

```
cd yolo
wget https://pjreddie.com/media/files/yolov3.weights
wget https://pjreddie.com/media/files/yolov3-tiny.weights

```

* Install requirements

```
pip install -r requirements.txt
```

## Run

```
# Yolo
python myLibYolo.py

# MobilenetSSD
python pedestrian_detector.py
```
